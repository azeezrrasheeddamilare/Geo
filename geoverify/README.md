# 🌍 GeoVerify Nigeria

Location-Validated Tokens on Solana - Built for Nigeria 🇳🇬

## 🚀 Features
- Verify real-world attendance at events
- Claim unique Geo-NFTs when physically present
- Fraud detection using AI
- Tourists can collect "Visit Nigeria" passport
- Event organizers can issue verifiable attendance proofs

## 🛠️ Tech Stack
- **Frontend**: React Native (Mobile), HTML/JS (Web)
- **Backend**: Node.js + Express
- **Blockchain**: Solana (Compressed NFTs)
- **AI**: Python (Fraud Detection)
- **Database**: PostgreSQL/Supabase
